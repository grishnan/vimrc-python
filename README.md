# 🧠 Vim Python Config (Full Featured `.vimrc`)

Этот репозиторий содержит готовую конфигурацию `~/.vimrc` для комфортной разработки на Python в Vim.
Подходит как для начинающих, так и для опытных пользователей, которым нужен быстрый и лёгкий IDE-подобный опыт без перехода на тяжёлые редакторы.

---

## 🚀 Возможности

### 🔌 Плагины

Используется менеджер плагинов **vim-plug**.

* `python-syntax` — расширенная подсветка Python
* `auto-pairs` — автоматическое закрытие скобок и кавычек
* `coc.nvim` — мощный движок автодополнения (LSP)

---

### ⚙️ Базовые настройки

* Нумерация строк + относительная (`number`, `relativenumber`)
* Умные отступы (`autoindent`, `smartindent`)
* Пробелы вместо табов (4 пробела)
* Быстрый отклик (`updatetime=300`)
* Подсветка совпадений скобок

---

### 🔍 Поиск

* Инкрементальный поиск
* Игнор регистра (с учётом smartcase)
* Подсветка результатов

---

### 🎨 Интерфейс

* Подсветка текущей строки
* Улучшенное меню автодополнения
* Всегда видимая статус-строка
* Колонка для знаков (ошибки, git и т.д.)

---

### 🐍 Python-специфика

* Ограничение длины строки: **79 символов**
* Автоматическая установка `textwidth` и `colorcolumn`
* Расширенная подсветка синтаксиса
* Подсветка встроенных функций и форматирования строк

---

### 🧪 Быстрый запуск и отладка

| Клавиша    | Действие                   |
| ---------- | -------------------------- |
| `F5`       | Запуск текущего файла      |
| `F6`       | Запуск с `pdb`             |
| `F7`       | Вставить `pdb.set_trace()` |
| `F8`       | Подсказка по pdb           |
| `Ctrl + P` | Быстрый запуск             |

---

### ⚡ Автодополнение (coc.nvim)

* `Tab / Shift+Tab` — навигация по списку
* `Enter` — подтверждение
* `Ctrl + Space` — вручную вызвать автодополнение

Дополнительно:

* `K` — документация (hover)
* `gd` — перейти к определению
* `[e` / `]e` — навигация по ошибкам
* `<leader>e` — список ошибок
* `<leader>f` — форматирование

---

### 🧹 Автоматизация

* Автоформатирование при сохранении
* Автоматическое добавление shebang:

```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-
```

* Подсветка лишних пробелов

---

## ⌨️ Лидер-клавиша

```vim
let mapleader = " "
```

Все пользовательские комбинации начинаются с **пробела**.
